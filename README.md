# 📊 Vrinda Store Sales Dashboard (Excel)

## 📌 Project Overview

This project presents an interactive **Excel Sales Dashboard** built using the Vrinda Store sales dataset. The dashboard provides valuable business insights into sales performance, customer behavior, order trends, and sales channels to support data-driven decision-making.

---

## 🎯 Objectives

- Analyze overall sales performance
- Compare Orders vs Sales
- Identify top-performing states
- Analyze customer demographics
- Compare Men vs Women purchases
- Analyze order status
- Evaluate sales channels
- Discover monthly sales trends

---

## 📈 Dashboard Insights

The dashboard includes the following visualizations:

- 📅 Orders vs Sales Trend
- 👨‍👩‍👧 Men vs Women Sales Comparison
- 📦 Order Status Distribution
- 🏆 Top 5 States by Sales
- 👥 Age Group & Gender Analysis
- 🛒 Sales by Channel
- 📊 Overall Sales Performance Summary
